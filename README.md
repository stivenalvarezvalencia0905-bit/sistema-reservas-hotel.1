# 🏨 Sistema de Reservas de Hotel

Sistema web para la gestión de reservas hoteleras desarrollado como proyecto formativo SENA.

## 🚀 Tecnologías
- Java 17
- Spring Boot
- JPA / Hibernate
- MySQL / H2
- HTML, CSS, JavaScript
- Fetch API

## 📌 Funcionalidades
- Login con roles (ADMIN / CLIENTE)
- Gestión de habitaciones
- Búsqueda avanzada por fechas
- Reservas y cancelaciones
- Disponibilidad automática
- Dashboard y agenda
- Correos simulados

## ▶️ Ejecución
1. Ejecutar el backend con Spring Boot
2. Abrir el frontend en el navegador
3. Acceder al sistema
